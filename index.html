INTEGRATED METRICS / 20 STATES

Case study / My contribution

From raw records to statistical evidence.

I analyzed 2,500 Phase 1 records, then helped integrate them with weekly CDC state data collected in Phase 2. My assigned analysis compared state death rates, visualized weekly trends for the highest-volume states, and measured relationships across the combined metrics.

Pipeline
Collect, clean, integrate, explore, test
Inference
Welch's t-test and Cohen's d
Result
p = 0.391, negligible effect (d = -0.038)
